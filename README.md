# ASO UNIDAD 2
## Entorno de Trabajo: Windows Server 2025

Herramienta Principal: Active Directory Domain Services (AD DS)

Durante este periodo, hemos trabajdo mucho con el despliegue de dominios y la administración centralizada de redes corporativas. Concretando nos quedaría estos 4 puntos:

### 1. Infraestructura de Dominio
Hemos trabajado en la base lógica de la red mediante la configuración de controladores de dominio.

Creación de Dominios: Instalación del rol AD DS y promoción del servidor para establecer nuevos bosques y dominios.

Configuración del Servidor: Ajustes iniciales de red y DNS necesarios para la funcionalidad del directorio activo.

### 2. Gestión de Clientes (Equipos)
Integración de los equipos de trabajo dentro de la infraestructura centralizada.

Vinculación al Dominio: Configuración de máquinas cliente para unirse al dominio creado.

Comprobación de conectividad: Verificación de la relación de confianza entre el cliente y el controlador de dominio.

### 3. Administración de Identidades (Usuarios y Grupos)
Gestión del ciclo de vida de los usuarios y optimización de la administración mediante agrupaciones.

Altas de Usuario: Creación de cuentas de usuario con sus respectivos atributos.

Gestión de Grupos: Creación de grupos de seguridad y distribución para simplificar la asignación de recursos.

Membresía: Asignación de usuarios a grupos específicos según roles o departamentos.

### 4. Estructura Jerárquica y Permisos
Organización lógica del directorio para aplicar políticas y delegar control de manera eficiente.

Unidades Organizativas (OU): Diseño de una estructura de OUs para segmentar objetos (por departamentos, ubicaciones geográficas, etc.).

Delegación y Permisos: Aplicación de permisos específicos basándose en la ubicación de los objetos dentro de las OUs, permitiendo una administración granular y segura.
